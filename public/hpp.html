Hosted Payment Page (HPP)

The fastest path — sign a redirect URL server-side and send the customer straight to Nuvei's hosted checkout page. Minimal server logic, no session token.

Lifecycle: Build redirect URL → Redirection → Return parameters → DMN ↺ Reset run
Amount: — Redirect built: no

Step 1Build the signed redirect URL

Computed server-side: SHA-256(secretKey + all parameter VALUES concatenated in submission order) — a different formula from the REST API's field-based checksum.

Step 3Return / output parameters

Nuvei can only redirect a real customer's browser to a publicly reachable return URL — unreachable from a local dev server. Use the simulate buttons to generate a correctly-signed return payload and validate it through the real checksum-verification code path.
